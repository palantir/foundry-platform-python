# FolderType

A folder can either a regular Folder, a
[Project](https://www.palantir.com/docs/foundry/getting-started/projects-and-resources/#projects) or a
[Space](/docs/foundry/security/orgs-and-spaces/#spaces).


| **Value** |
| --------- |
| `"FOLDER"` |
| `"SPACE"` |
| `"PROJECT"` |
